# kampus official website

###### Color palette

```
—→ primary color: RGB(11, 37, 48)
—→ secondary color : RGB(230, 232, 236)
—→ accent color: RGB(55, 160, 169)
representing a sleek, professional, and exclusive brand identity for Kampus Agency
```

###### Audiences
```
1. tech-savvy business and startups looking for professionalism, cutting-edge design, user-friendly platforms, and reliability.
2. creative agencies and designers looking for High-level design, seamless integration, and collaboration between creativity and technology.
```
